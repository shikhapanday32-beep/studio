# **App Name**: StudyZen

## Core Features:

- Task Creation: Allow users to add daily study tasks with subject name, task title, and estimated time in hours.
- Task Completion: Enable users to mark tasks as complete.
- Task Deletion: Allow users to delete tasks from their study plan.
- Total Study Hours: Display the total planned study hours for the day.
- Data Persistence: Persist study plan data using local storage.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) for a calming and focused environment.
- Background color: Very light blue (#F0F8FF) for a clean and distraction-free interface.
- Accent color: Light green (#90EE90) to indicate task completion and important actions.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable experience.
- Mobile-first, responsive layout that adapts to different screen sizes for optimal usability.
- Subtle transitions and animations to provide visual feedback on user interactions.
